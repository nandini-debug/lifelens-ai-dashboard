# 🌌 LifeLens AI — Power BI Dashboard

A futuristic **Digital Lifestyle Intelligence Dashboard** built using Power BI to analyze productivity, digital habits, burnout patterns, sleep behavior, and lifestyle insights through interactive visualizations.

---

## 🚀 Dashboard Overview

LifeLens AI helps visualize behavioral and productivity trends using modern analytics and a premium dark-themed dashboard UI.

The dashboard focuses on:

- 📱 Screen Time Analytics
- 😴 Sleep Pattern Insights
- 🧠 Focus Trend Analysis
- 🔥 Burnout Risk Monitoring
- ⚡ Productivity Tracking
- 🍩 Digital Activity Distribution
- 📊 AI Behavioral Insights
